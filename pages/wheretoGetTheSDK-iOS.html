Where to get the SDK?

Currently, you can get it through SPM only using the following GitHub link (Please make sure to always use the latest version):
GitHub - Tamatem-Plus/iOS-SDK

Supported Versions

Tamatem Plus iOS SDK works with Swift 4.2 and later and iOS 12 and later.

Please make sure that your app deployment target is iOS 12 or later.
Currently, we are not supporting direct access of Objective-C code to the Interface.
