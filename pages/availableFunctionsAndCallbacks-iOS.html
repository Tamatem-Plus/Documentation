Available Functions & Callbacks:

Tamatem Plus SDK allows you to connect your users with the Tamatem ID through our public functions. The following functions can be used by accessing the singleton instance
“TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance?”:

  1. connect: to connect the user to Tamatem Plus. You can use the callback function onConnected(didSucceed: Bool) to handle connection success or failure status. You can also use the callback function onRedirectedToApp() to place any logic needed when the user is redirected to the app during the connection process.
  2. checkConnectionStatus: Call checkConnectionStatus(completion: { success in … to check if the user is connected or not. The status returned in the callback.
  3. getInventoryItems: Call getInventoryItems(completion: { inventoryItems in … to get All the items purchased by the user on Tamatem Plus (redeemed items and unredeemed items).
  4. getInventoryItems(isRedeemed: Bool): Call TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance?.getInventoryItems(isRedeemed: true, completion: { inventoryItems in … to get the items purchased by the user on Tamatem Plus. Send “true” to get the redeemed items, and “false” to get the unredeemed items.
  5. redeemAllItems: Call TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance?.redeemAllItems(completion: {success in … to redeem all items.
  6. getUserInfo: Call TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance?.getUserInfo(completion: {success, user in … to get the user data from Tamatem Plus.
  7. logout: Call TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance?.logout(completion: { success in … to clear the Tamatem user session and clear the stored data on the device (Please note that this will NOT delete the stored data from Tamatem Plus records).
You can find an example script to check out how these functions are used in the GitHub release.
