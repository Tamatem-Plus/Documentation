Getting started

iOS SDK provides a seamless way to connect Tamatem ID, list Purchased items, and redeem them in your iOS app.

How to Install

We only support the Swift Package Manager method for installation.
Our GitHub repository provides a compatible package for SPM.

Installation Steps:
  1. Get our SDK Package link from the mentioned repository.
  2. Open your project and select file->Add package dependencies.
  3. Paste the SDK link in the search/Package URL box.
  4. Click on Add Package.
