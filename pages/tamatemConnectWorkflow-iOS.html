TamatemConnect Workflow? How TamatemConnect works?

This is a brief look at the workflow

  1. Call the public function “TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance.Connect()”.
  2. The public function call function at the TamatemConnect Plugin.
  3. The plugin requests the service from the Tamatem Plus Portal.
  4. Tamatem Plus Portal responds with state or retrieved data.
  5. when the user returns to the app you should call “TamatemPlusiOSFramework.TamatemConnectPlugin.sharedInstance.handleOAuthCallback(url: incomingURL.absoluteString)” to pass the received data to the plugin
  6. The TamatemConnect plugin sends back the state or retrieved data.
  7. You can now use the other SDK functions.
