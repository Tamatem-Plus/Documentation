TamatemConnect Workflow? How TamatemConnect works?

  1. Call the public function “TamatemConnectPlugin.Instance.Connect()”.
  2. The public function calls a function at the TamatemConnect Plugin.
  3. The plugin requests the service from Tamatem Plus Portal.
  4. Tamatem Plus Portal responds with state or retrieved data.
  5. TamatemConnect plugin sends back the state or retrieved data.
  6. You can now use the other SDK functions.
